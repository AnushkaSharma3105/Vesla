Overview

Vesla is an IoT-based water quality monitoring system designed to provide real-time insights into household water safety. It uses smart sensors and a web interface to detect contamination and alert users instantly.

This system helps households monitor water quality in tanks, pipelines, and RO systems without manual testing.
